## Dataset

This project uses the IBM Telco Customer Churn dataset, publicly available on Kaggle.

Due to dataset licensing considerations, the raw data is not included in this repository.

You can download the dataset from:
https://www.kaggle.com/datasets/yeanzc/telco-customer-churn-ibm-dataset

After downloading, place the CSV file in the `data/` directory before running the notebook.

The dataset is used for educational and non-commercial purposes in accordance with Kaggle and IBM sample data usage guidelines.
